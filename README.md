# Prerequisites:
- NodeJS v8.7.0
- npm v5.4.2+ / Yarn v1.2.1 

# Development guide


### Install packages

```$ npm install```

For Yarn:

```$ yarn```

### Run in development mode

```npm run start```

For Yarn:

```yarn start```

### Build for production

```npm run Build```

For Yarn:

```yarn start```

### Running express server 

```npm run prod```

For Yarn:

```yarn prod```

Then open `http://localhost:3000` in browser.
